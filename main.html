Question-3: What is event delegation in the DOM?

Answer: Event delegation is a technique in JavaScript where an event listener is attached to a parent element instead of adding multiple listeners to child elements. where events are child to parent elements.

Question-4: What is event bubbling in the DOM?

Answer: Event bubbling is a phase of event propagation where an event starts at the target element and moves up the parent elements of the DOM.
